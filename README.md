This Git respository is purpose for MRTB2173 case study which our topic of "Predicting Surgical Procedure Duration to Support Operating Room Scheduling Decisions."

The files included:
1.Train_Model_v1 = Model v1 baseline prediction model.
2.Train_Model_v2 = Model v2 improved/enriched prediction model.
3.predictive_app = A predictive app that provide UI for user input and connected to both Models to calculate the predictive result.
4.monitor_dashboard = A monitor dashboard that provide UI for internal user to review the predictive app's user feedback and overall feedback statistics.
